## Partes importantes do Exercício

Estrutura:

- `src`

    - `Homem`: Classe que extend PessoaIMC.
    - `Mulher`: Classe que extend PessoaIMC.
    - `Pessoa`: Classe Pessoa.
  - `Main`: Classe main do Exercício.
